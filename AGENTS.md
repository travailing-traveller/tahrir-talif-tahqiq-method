# AGENTS

This repository contains the Taḥrīr / Taʾlīf / Taḥqīq Method Kit.

Before editing the kit, read:

1. `README.md`
2. `method-philosophy.md`
3. `FEEDBACK.md`
4. `method-kit/`
5. `project-scaffold/README.md`

Do not weaken the core guardrails without documenting the reason.

Key principles:

- source sovereignty;
- retrieval-grounded source use;
- unit-by-unit verification;
- role separation;
- model agnosticism;
- confidence tags;
- preserved uncertainty;
- human responsibility.
