# 06 — Register Distillation

Use this only after a verified base unit or verified set of units exists.

## Purpose

Transform verified material into a defined register without smuggling in new source-claims.

## Possible registers

- academic prose;
- annotated edition;
- teaching handout;
- student-facing explanation;
- lecture notes;
- sermon or exhortation;
- public-facing summary;
- Arabic scholarly prose;
- legal memorandum;
- publication copy.

## Rules

- State the source units used.
- State the register and audience.
- Do not introduce new source-claims.
- Mark application or reflection as application or reflection.
- Preserve known uncertainties.
- Maintain terminology policy.

## Output note

Every register output should include:

- source units;
- register;
- intended audience;
- transformation constraints;
- verification status;
- cautions.
