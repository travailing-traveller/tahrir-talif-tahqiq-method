# 05 — Taḥqīq Verification

Use this to verify one completed unit. Ideally, use a different model or reviewer from the one that performed Taʾlīf.

## Verify

- source spans;
- source handles;
- exact or normalised matches;
- translation fidelity;
- omissions;
- additions;
- terminology;
- citations;
- claims;
- uncertainty handling;
- register boundaries.

## Classification

Classify the unit as one of:

- `VERIFIED`
- `VERIFIED WITH NOTES`
- `REQUIRES REVIEW`
- `UNVERIFIED`
- `REJECTED`

## Confidence tags

Use tags where appropriate:

- `verified-exact`
- `verified-normalised`
- `verified-translation`
- `paraphrase`
- `editorial-judgement`
- `application`
- `unverified`
- `requires-review`
- `rejected`

Do not mark a unit verified if important questions remain unresolved.
