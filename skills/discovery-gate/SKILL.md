# Discovery Gate Skill

Use when a project has not passed discovery.

Ask for a one-line brief first. Then interview rigorously. Ask one cluster at a time. Challenge vague answers. Do not proceed if the project is not sufficiently thought through.

Gate statuses:

- `NOT READY — insufficient thought`
- `PARTIALLY READY — more discovery needed`
- `READY FOR TAḤRĪR FORMULATION`
