# Retrieval Grounding Skill

Use whenever source text, attribution, citation, vocalisation, or primary material is involved.

Rules:

- Do not originate primary text from memory.
- Do not originate vocalisation from memory.
- Do not originate attribution, isnād, edition data, or page references from memory.
- Every quoted span must have a source handle.
- If material cannot be retrieved, place it in uncertainty, not final output.
- If an external reference is used, record it.
